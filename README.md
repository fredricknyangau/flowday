# Flowday

A calm day planning system built for academic freelancers managing
multiple clients, same-day deadlines, and overnight work sessions.

## Status
> Phase 0 — Project setup and documentation

## Docs
- [Product Requirements Document](docs/PRD.md)
- [Database Schema](docs/SCHEMA.md)
- [API Reference](docs/API.md)

## Stack
- Frontend: React + TypeScript + Tailwind CSS (PWA)
- Backend: FastAPI + Python
- Database: PostgreSQL

## Phases
- **MVP** — Assignment tracker, deadline countdown, daily schedule view
- **Phase 2** — Client tracker, weekly overview, push notifications
- **Phase 3** — Learning block, income tracker, burnout warning
